# Perplexity MCP for Cursor (recommended)

## Why

This enables up-to-date web research inside Cursor Agent workflows (docs, CVEs, best practices).

## Official docs

- Perplexity MCP server: `https://docs.perplexity.ai/guides/mcp-server`

## Cursor config

This playbook installs a ready-to-edit config at:

- `~/.cursor/mcp.json`

Edit the value:

- `PERPLEXITY_API_KEY`

Then restart Cursor.

## Notes

Perplexity provides a one-click installer for Cursor in their docs. See the “Install in Cursor” section:

- `https://docs.perplexity.ai/guides/mcp-server`
