# Workflow Macro: Implement Task (TDD)

---

Implement the next task using TDD.

## Inputs
- Task id/title
- Acceptance criteria checklist
- Files to create/modify
- Test commands

## Steps
1. `/test-writer`: write/update tests for all acceptance criteria; run tests and confirm failure.
2. `/implementer`: implement minimal code to pass tests; run tests again; refactor only if needed.

## Output
- What changed (files)
- Tests run + results
