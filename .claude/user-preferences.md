# User Preferences

Persistent preferences for this user across all sessions.

## Shell

- **Shell**: Fish shell
- **Venv activation**: `source .venv/bin/activate.fish`
- **Commands**: Use Fish-compatible syntax (no bash-isms like `export VAR=value`, use `set -x VAR value` instead)

## Environment

- **Python**: Use `.venv/bin/python` for commands
- **Tests**: `.venv/bin/python -m pytest tests/ -v`
